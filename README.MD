# Cloudflare 优选 IP

**测试地点**: 中国河北电信  
**更新频率**: 每天7点、13点更新  
**本次更新**: 2026-05-10 18:01:13

## 订阅链接

- 高速优选：`https://raw.githubusercontent.com/weifengjia/czctip/refs/heads/main/best_ips.txt`
- 所有可用：`https://raw.githubusercontent.com/weifengjia/czctip/refs/heads/main/full_ips.txt`

## 使用方法

1. 复制订阅链接。
2. 打开 EdgeTunnel 后台。
3. 创建优选订阅，选择“优选订阅模式 - 自定义”。
4. 在订阅接口 / API / URL 中粘贴订阅链接。
5. 验证可用性，选择“追加 API”。
6. 保存并重新订阅。
